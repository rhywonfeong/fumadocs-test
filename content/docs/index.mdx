帮咱解决 heroui pro v3 使用问题的，包括 pro 依赖的安装和 ai tools 的使用。

非账号、官 key 共享（官方有防控措施，风险太大），走的是代理。

也就是：

- pro 依赖安装的时候，在 worker 替咱下载最新的 tarball，自动装配到项目，让 `@heroui-pro/react`、`heroui-native-pro` 这类依赖导入可用（CI/CD 环境也行）
- mcp 初始化和工具调用的时候，用官方发的 token 提咱请求上游，然后原样返回。

先从 hpsetup 开始，完整安装 pro 依赖（非空壳），然后再使用 mcp & skills 辅助开发。

hpsetup 需要一个 hp key，mcp 和 skills 需要一个 personal token。两者不同，可以在下边统一配置，之后所有页面都会内嵌真实的 key/token，方便复制。
