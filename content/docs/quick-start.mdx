hpsetup 是 HeroUI Pro 组件库的安装工具，支持 React 和 React Native 平台。

## 前置条件

- Node.js 18+
- npm / pnpm / yarn / bun（不支持 Yarn Berry PnP）
- 有效的 API Key（`hp_` 前缀）

## 安装

```bash
npx -y hpsetup@latest <your-key>
```

首次运行时，hpsetup 会引导你完成交互式配置：
选择产品（React Pro 或 Native Pro）→ 检测已有安装 → 下载组件 → 安装 peer 依赖。

## 支持的产品

| 产品 | 包名 | 产物目录 | 伴随包 |
|------|------|---------|--------|
| HeroUI React Pro | `@heroui-pro/react` | `dist/` | `@heroui/react` |
| HeroUI Native Pro | `heroui-native-pro` | `lib/` | `heroui-native` |

## 非交互模式

在 CI 或脚本中使用时，设置 `HEROUI_KEY` 环境变量即可跳过交互：

```bash
HEROUI_KEY=hp_xxxx npx -y hpsetup@latest
```

## 包管理器

hpsetup 自动检测当前项目使用的包管理器并使用对应命令安装 peer 依赖：

- **npm** — `npm install --legacy-peer-deps`
- **pnpm** — `pnpm add`，并自动将 peer 依赖加入 trust allowlist
- **yarn** — `yarn add`（仅限 Classic，不支持 PnP）
- **bun** — `bun add`，并自动管理 trust allowlist

## 本地缓存

下载的 tarball 缓存在 `~/.heroui/cache/<product>/<version>/`，相同版本不会重复下载。

使用 `--no-cache` 可强制从源站拉取最新包（受每日频率限制）。
